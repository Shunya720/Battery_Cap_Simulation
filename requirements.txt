# README.md
# バッテリー容量別シミュレーション比較システム

複数のバッテリー容量で需要平準化効果を比較し、最適容量を検討するためのWebアプリケーションです。

## 機能
- 需要予測データのCSVアップロード
- 複数容量での自動最適化シミュレーション
- 容量別効果比較とグラフ表示
- 結果のCSVダウンロード

## 使用方法
1. 需要予測データ（96ステップ、15分間隔）をCSVでアップロード
2. 比較したい容量を設定
3. シミュレーション実行
4. 結果を確認・ダウンロード

## 技術スタック
- Streamlit
- Pandas, NumPy
- Plotly
- Python 3.9+

## ローカル実行
```bash
pip install -r requirements.txt
streamlit run battery_capacity_app.py
```
